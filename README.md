# battery_test_project
Battery_test_project is a project that use lab instrument to test the LiPo battery.
* OS
    * Windows 11
* API
    * NI-VISA
* Lab instrument
    * TEXIO PDS20-36A
    * Rigol DL3021
    * Keysight DAQ970A
    * DELTA DVP-12SE
## Required Libraries
* pyVISA
* time
* serial
* pandas
* matplotlib
